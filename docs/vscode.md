# Notes for VS-Code

Optional **Syntax Highlighting** present.

## Steps to add Syntax Highligting

* Press `Ctrl + Shift + P`.
- Select `Developer: Install Extension From Location`.
- On the pop-up select `compy-vscode`.
- Again Press `Ctrl + Shift + P`.
- Select `Developer: Reload Window` this time.
- Open a `.cmpy` file and enjoy _syntax highlighting_.
